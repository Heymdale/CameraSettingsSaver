### 🇺🇸 English

1. This folder contains files for testing the compiled application: `.ods` tables with a list of actions, `link_creator.bat` for creating shortcuts, `.bat` files for launching the application with required parameters, and `.json` files necessary for certain checks.
2. This `test` folder must be located in the compiled application's directory, at the same level as the `WebCamSettings.exe` file.
3. First, run `link_creator.bat` from this folder — it will create the missing shortcuts.
4. Each `.bat` or `.lnk` file corresponds to a row in `Tests.ods`.

---

### 🇷🇺 Русский

1. В данной папке представлены файлы для проверки скомпилированного приложения: таблицы `.ods` с перечнем действий, `link_creator.bat` для создания ссылок, `.bat`-файлы для запуска приложения с нужными параметрами, `.json`-файлы, необходимые для некоторых проверок.
2. Данная папка `test` должна находиться в папке скомпилированного приложения на одном уровне с файлом `WebCamSettings.exe`.
3. В первую очередь запустите `link_creator.bat` из данной папки — он создаст недостающие ссылки.
4. Каждый `.bat` или `.lnk` файл соответствует строке из `Тесты.ods`.
